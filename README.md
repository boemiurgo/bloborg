## Bloborg – basic one-page site

Single-page HTML (no build tools) with:
- Sticky top bar (logo + links)
- Hero section with CTA
- Prices section (grid/table-like layout)
- “Get a quote” contact section (form)

### Run
Open `index.html` in a browser.

### Replace assets
Put your logo and reference images into `assets/` and update paths in `index.html`.
